按钮系列(buttons)

可作为按钮使用的标签或元素,一般为<a><button><input>标签，通过样式.hm-btn来定义。

基本使用

基础样式

在要应用按钮样式的元素上添加 .hm-btn，再设置相应的颜色；考虑到移动端的布局特性，按钮默认显示为块级元素，设置了display:block。

  • .hm-btn - 基础按钮，无样式
  • .hm-btn-default - 默认，带边框灰色按钮
  • .hm-btn-selected - 已选择按钮
  • .hm-btn-link - 链接按钮
基础按钮 默认按钮 已选择按钮 链接按钮

按钮颜色

在.hm-btn上追加颜色样式。

  • .hm-btn-primary - 首选，蓝色按钮
  • .hm-btn-success - 成功，绿色按钮
  • .hm-btn-info - 一般信息，浅蓝色按钮
  • .hm-btn-warning - 警告，橙色按钮
  • .hm-btn-danger - 危险，红色按钮
首选按钮 成功按钮 信息按钮 警告按钮 危险按钮 黑色按钮

按钮大小

在.hm-btn上追加大小样式。

  • .hm-btn-xs - 加小按钮
  • .hm-btn-sm - 小按钮
  • .hm-btn-lg - 大按钮
  • .hm-btn-xl - 加大按钮
加小按钮 小按钮 正常按钮 大按钮 加大按钮

按钮形状

在.hm-btn上追加形状样式。

  • .hm-radius - 圆角按钮
  • .hm-round - 椭圆形按钮
默认为直角 圆角按钮 椭圆形按钮

按钮状态

在.hm-btn上追加状态样式。

  • .hm-disabled - 不可用
不可用按钮

空心按钮

通过给按钮添加 .hm-btn-hollow类，可使按钮成为空心按钮。

首选按钮 成功按钮 信息按钮 警告按钮 危险按钮

行内按钮

按钮默认显示为块级元素，通过给按钮添加 .hm-btn-inline类，按钮也变为了行内（inline-block）元素,而且宽度也会变成auto。
