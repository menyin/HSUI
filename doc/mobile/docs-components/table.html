表格状态

通过这些状态类可以为行或单元格设置颜色。

  • .hm-active 激活；
  • .hm-disabled 禁用；
  • .hm-primary 标识普通的提示信息或动作；
  • .hm-success 标识成功或积极的动作；
  • .hm-warning 标识警告或需要用户注意；
  • .hm-danger 标识危险或潜在的带来负面影响的动作。
