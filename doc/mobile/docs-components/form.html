表单布局

注意：早期版本表单布局结合hm-listview类进行布局，现已不采用该方式
在容器上添加 .hm-form类，进行表单的布局。所有添加了 .hm-form-control 类容器下的的 <input>、<textarea> 和 <select> 元素都将被默认设置宽度属性为 width: 100%;。 将 label元素和前面提到的hm-form-control控件包裹在 .hm-form-group 中可以获得最好的排列。

左右结构表单布局

表单控件布局使用.hm-form-group类，默认为左右结构的表单布局。
