列表(listview)

列表通过样式.hm-listview来定义。属于移动端使用较为频繁的组件之一。

请注意

基于kendoUI实现列表样式时，需要把整个完整标签都放在kendouUI原有的LI标签里面。 

    
    HSUI的标签结构：
    <div class="hm-li">...</div>
    
    基于KENDOUUI的标签结构：
    <li>
        <div class="hm-li">...</div>
    </li>

基本样式

列表项类型

给列表项，添加类.hm-li-static可以快速创建一个带有只读列表项的列表。

列表分隔
静态列表
带跳转的列表，如果右侧不想带箭头可以不写hm-li-link

列表项结构

列表项默认为线性行布局，由三部份组成（可选）,hm-li-header hm-li-content hm-li-footer,并需要指定某一部份为主要部份并添加hm-flex-1样式。

header
content
footer 
<div class="hm-listview">
     <div class="hm-li">
    	<div class="hm-li-header">...</div>
        <div class="hm-li-content">...</div>
        <div class="hm-li-footer">...</div>
    </div>
</div>

情景列表

列表项默认为水平线性布局，由三部份组成（可选）,hm-li-header hm-li-content hm-li-footer,并需要指定某一部份为主要部份并添加hm-flex-1样式。

hm-li-success
hm-li-info
hm-li-warning
hm-li-danger

有序列表

列表前带有序号。

  1. 分隔
  2. 列表项标题
  3. 列表项标题

列表项可设置为列布局

列表项通过追加 hm-li-column样式，可实现列布局，支持多行复杂内容结构。
