反馈信息(feedback)

对于用户的操作给于反馈，并提示相应的处理。一般用于弹窗或者整个页面区域的操作反馈。

默认反馈

结合弹窗控件使用，内容水平居中。

 
你是否要执行该操作
执行该操作将带来不可逆的后果，请再次确认你是否一定要执行该操作？
取消 确定

带提示图标的对话窗

能过hm-feedback-img-图标名来定义反馈图标

  •  
    恭喜您，申请成功了！
    您已成功申请了一张烟叶会员卡

    取消 确定
  •  
    对不起，申请失败！
    您不能申请烟叶会员卡，你需要先成为会员，然后才能再次申请！
    取消 继续

带卡通人物的场景提示

通过hm-feedback-cartoon-图标名来定义反馈卡通。

  • 恭喜您，申请成功了
    您已成功申请了一张烟叶会员卡
  • 加载失败
    请点击屏幕重新加载
  • 暂无数据
    请重新输入查询条件查询
  • 努力加载中
    正为为您加载数据，请稍候...
  • 网络不给力哦
    请检查网络设置后点击刷新

带css3动画的场景提示

通过hm-feedback-animation-图标名来定义反馈动画。
