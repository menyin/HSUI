图片(images)

定义图片样式。

响应式图片

通过为图片添加 .hm-img-responsive 类可以让图片支持响应式布局。其实质是为图片设置了 max-width: 100%;、 height: auto; 和 display: block; 属性，从而让图片在其父元素中更好的缩放。

图片背景

通过为图片添加 .hm-img-bg 类可以以背景图片的方式显示缩略图，无论宽高比例如何都可以保证图片不变形。

增强样式

图片圆角

为<img>元素设置不同的 class，增强其样式。

  • .hm-radius 圆角
  • .hm-round 椭圆，因为效果不是太好看，比较少用
  • .hm-circle 圆形，一般用于正方形的图片(你要觉得椭圆好看，用在长方形上也可以)
