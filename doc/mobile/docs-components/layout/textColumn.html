文本多列布局(textColumn)

文本多列布局：通过.hm-textColumn能够创建多个列来对文本进行布局 - 就像报纸那样。

通过.hm-textColumn对容器内的长文本实现多列布局,通过给内部元素添加.hm-column-span-all来实现该部份内容横跨所有列。默认文本多列布局为三列,间隔40px,1PX的间隔线

据历史文献记载，我国的烟草是16世纪相继由菲律宾、越南、朝鲜传入，很快为人们所接受。经过四百余年的种植和传播，烟草在我国的分布已经十分广泛,东起黄海之滨，西至伊犁地区，南起海南岛，北迄白山黑水间，在经度74°－135°，纬度80°—53°范围内，从低于海平面的吐鲁番盆地到海拔3000M以上的高原山区,到处都留下了烟草的足迹。在明清以来的古籍中，关于烟草的称谓可谓五花八门,有将烟草称为“淡巴菰”、“但不归”的，也有以其色香赋予诸如芬草、仁草、金丝草、金丝烟、金丝醺、还魂草、神草、相思草之类雅名的。在长期的流传和发展中,人们对烟草的称谓最终定格为现今的“烟”。目前人们普遍认为烟草最早源于美洲。考古发现，人类尚处于原始社会时，烟草就进入到美洲居民的生活中了。那时，人们在采集食物时，无意识地摘下一片植物叶子放在嘴里咀嚼，因其具有很强的刺激性，正好起到恢复体力和提神打劲的作用，于是便经常采来咀嚼，次数多了，便成为一种嗜好。
