表格布局(tableLayout)

表格布局：是一个容器以行列的形式显示子容器元素，通过.hm-tableLayout定义容器为表格布局，通过.hm-table-row来定义表格行,通过.hm-table-cell来定义单元格，一格为一列。

实例：基础表格布局

通过.hm-tableLayout来定义表格布局。默认表格布局宽度100%，无边框，单元格无间距

 <div class="hm-tableLayout" >
    	<div class="hm-table-row">
            <div class="hm-table-cell">单元格</div>
            ...
        </div>
        ...
    </div>
