缩略图(Thumbnail)

通过缩略图组件结合布局系统，可以很容易地展示出栅格样式的图像、视频、文本等内容。

默认样式

hm-thumbnail类可直接用于div或者a元素


    <ul class="hm-average-4" >
      <li>
        <a href="#" class="hm-thumbnail">
          <img src="img-big.jpg" alt=""/>
        </a>
      </li>
    </ul>

自定义内容

添加一点点额外的标签，就可以把任何类型的 HTML 内容，例如标题、段落或按钮，加入缩略图组件内。
