导航(navs)

导航样式组件，在 <ul> 链接列表中添加 .hm-nav class。

基本样式

垂直导航(默认样式)

<ul> 添加 .hm-nav class 以后就是一个基本的垂直导航。默认样式中并没有限定导航的宽度，可以结合网格使用。

在 .hm-nav 的基础上再添加 .hm-nav-pills，形成一个水平导航。

标签式导航

默认为标签在顶部的效果

在 .hm-nav 的基础上添加 .hm-nav-tabs，形成一个标签式的导航。激活的标签在 <li> 上添加 .hm-active。

  • 标签页

标签在底部

在 .hm-nav 的基础上添加 .hm-nav-tabs-bottom，形成一个标签式的导航。激活的标签在 <li> 上添加 .hm-active。

通过 .hm-nav-justified 类可以很容易的让标签页或胶囊式标签呈现出同等宽度。

标签页的图标，主要是通过标签页导航按钮添加图标来实现。

图标在左侧
