图标(icons)

图标类型

字体图标(iconfont)

图标字体化，字体图标除了图像清晰度之外，图标字体可以用过font-size属性设置其任何大小，还可以加各种文字效果，包括颜色、Hover状态、透明度、阴影和翻转等效果。图标字体兼容IE8及大部份的现代浏览器。

  • 图标字体只能被渲染成单色或者CSS3的渐变色。
  • 建议统一通过 http://www.iconfont.cn/ 平台进行图标的制作及管理

注：考虑到制作成本，暂时未采用该类型图标。

图片图标

传统的图片图标。

基本使用

使用方法

在元素标签上添加 hm-icon-{图标名称} class。

图标颜色

字体图标的颜色取决于字体颜色，所以在字体图标元素标签上追加字体颜色样式 hm-color-{颜色名称}可改变图标颜色。

  • .hm-color-muted
  • .hm-color-primary
  • .hm-color-info
  • .hm-color-success
  • .hm-color-warning
  • .hm-color-danger

图标大小

图标默认为24X24PX大小，在图标元素标签上追加hm-icon-{尺寸}可调整图标大小。

  • .hm-icon-sm，显示为16X16
  • .hm-icon-lg，显示为32X32
  • .hm-icon-xl，显示为48X48

图标形状

形状类型

在元素标签上添加形状样式hm-{形状名称}。

  • .hm-radius，显示为圆角方形
  • .hm-circle，显示为圆形

形状+尺寸

在元素标签上添加形状样式hm-{形状名称}及尺寸样式hm-icon-{尺寸}。

  • hm-radius hm-icon-lg，显示为圆角方形大图标
  • hm-circle hm-icon-xl，显示为圆形加大图标

形状+背景颜色

在元素标签上添加形状样式hm-{形状名称}及尺寸样式hm-bg-{背景颜色}。

  • hm-circle hm-icon-lg hm-bg-primary，显示为圆形的蓝色大图标
