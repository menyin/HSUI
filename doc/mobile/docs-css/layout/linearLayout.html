线性布局(linearLayout)

线性布局：是一个容器以线性方向显示它的子容器元素，分为垂直布局和水平布局。线性布局是让子容器针对父容器的宽、高度按一定规则进行划分，来实现布局的垂直等高、水平均分、按比例划分等。

水平布局

通过给父容器添加.hm-linearLayout样式来实现，内部子容器的水平方向的平均分、按比例划分、固定宽及按比例划分混合布局。

实例：水平均分

通过设置子容器的style="-webkit-box-flex: 1;"来实现平均或者按比例显示宽度。注：如果子容器内容为长文本时，将无法实现自适应效果，解决的办法是设置宽度为width:1%;
