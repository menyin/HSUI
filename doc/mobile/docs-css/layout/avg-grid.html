等分网格(Average Grid)

线性布局：Average Grid，通过使用 ul / ol 创建等分列，用于内容的排列。

基本使用

只添加 .hm-average-*,这里的数字表示几等分，最多支持12等分，比如 .hm-average-4 会将子元素< li > 的宽度设置为25%。

<ul class="hm-average-4" >
    	<li>内容</li>
    	<li>内容</li>
    	<li>内容</li>
    	<li>内容</li>
</ul>
