目录结构

以下是HSUI所提供的文档目录结构。 

            	
HSUI
├── resources
│     ├── css
│     │     ├── images             // CSS里用到的所有图片
│     │     │     ├── alert
│     │     │     ├── form
│     │     │     ├── framework
│     │     │     └── icons
│     │     ├── animate.css       // CSS3动画库
│     │     ├── base.css          // HSUI 所有样式文件
│     │     ├── global.css        // HSUI 所有样式文件
│     │     ├── icon.css          // 图标
│     │     └── normalize.css      // 统一样式
│     ├── images                   // 页面上插入的图片
│     │     ├── app-icon72x72@2x.png
│     │     └── favicon.png
│     └── js
│            ├── jquery-1.9.1.js    // HSUI所有的插件都是基于jquery
│            └── hsui.js            // 插件JS或者前端人员自已编写的JS
├── framework-grid.html
├── framework-slider.html
└── page-alert-loading.html

HSUI 插件全部依赖 jQuery

请注意，HSUI 的所有 JavaScript 插件都依赖 jQuery，因此 jQuery 必须在 HSUI 之前引入。 HSUI目前所支持的 jQuery 版本为jquery-1.9.1。
