布局

HSUI定义的一些基础样式。

浮动布局(class="clearfix")

通过float:left/float:right来实现标签内容居左/居右布局,浮动布局，必需利用.clearfix来清除浮动

  • h-row h-text-center：表示采用浮动布局，加这一样式的目的是为了实现中间栏居中布局
  • float-left：浮动到左侧
  • float-right：浮动到右侧
  • clearfix：用来清除浮动

左右固定布局

布局中各个栏宽度不固定，而且也不按一定的比例划分，最常见的是工具栏的左右布局

  • h-relative：如果父标签已定义了positon:absolute,或者position: relative，那么不需要加这个样式
  • h-pull-left：绝对定位到左侧
  • h-pull-right：绝对定位到右侧

定宽多栏布局

是为了实现，多栏布局其中一栏的宽度是自适应的，其它栏宽度固定的布局，主要提供两栏和三栏布局，如果超过三栏，可以采用两栏、三栏以及结合栅格布局系统进行多层嵌套

定宽两栏布局(class="h-row h-row-col2-rf")

是为了实现,布局分左右两栏,其中一栏的宽度是自适应的，另一栏宽度固定的布局

  • col2：Column 2,表示两栏布局
  • lf：left fluid,左边自适应
  • rf：right fluid,右边自适应
