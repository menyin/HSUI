Border布局

它将整个容器分为5个部分，分别是：east, south, west, north, center。加入到容器中的子面板需要指定要加入到那个部分，这5个部份并非一定要全部都有，但最少要有center部份。border布局采用position: absolute来布局，所以其父容器必须指定position: relative;，另外所有的布局都需要考虑到各个部份之间的间隔。程序实现时可以通过设定一个间隔参数。

每一个部份的style都应具有left,right,top,bottom,height,width,这几个值，只是有些值是需要开发人员指定，而有些值则是通过程序计算出来的

开发人员指定参数

  • north：最少要指定height
  • south： 最少要指定height
  • west： 最少要指定width
  • east： 最少要指定width
  • center： 不需要指定任何值，由程序计算
  • 间隔参数space: 可以不指定，会取默认值5px

程序计算部份

  • north： left(默认为0+space)，right(默认为0+space)，top(默认为0+space)，bottom(不需要指定，取默认值auto)，width:(不需要指定，取默认值auto)
  • south： left(默认为0+space)，right(默认为0+space)，top(不需要指定，取默认值auto)，bottom(默认为+space)，width:(不需要指定，取默认值auto)
  • west：left(默认为0+space)，right(不需要指定，取默认值auto)，top(north.height+space*2)，bottom(south.height+spaceX2)，height:(父容器.height-north.height-south.height-space*3)
  • east：left(t(不需要指定，取默认值auto)，rig默认为0+space)，top(north.height+space*2)，bottom(south.height+spaceX2)，height:(父容器.height-north.height-south.height-space*3)
  • center：left,right,top,bottom,width,height

注意：多层嵌套时，从第二层开始上下左右四个space要去掉，父容器height:auto,在IE7下，子容器设置百分比，是无法获取到高度的，所以布局中各个容器的高度都要计算出具体的值。
